# Сравнительный анализ сервисов геокодирования

## Описание

Этот репозиторий содержит результаты сравнительного анализа трёх сервисов геокодирования: **Яндекс Геокодер**, **DaData** и **GigaChat (Сбер)**. Анализ проводился для оценки точности и успешности прямого и обратного геокодирования на тестовой выборке адресов.  

Цель репозитория — помочь выбрать подходящий сервис геокодирования в зависимости от точности, возможностей API и бюджета.

---

## Содержание репозитория

- `dataset-geo.xlsx` — исходный набор адресов для тестирования.  
  [Скачать датасет](https://github.com/myaccforedu-hash/geocoding-api-comparison-analysys/blob/main/dataset-geo.xlsx)

- `result/` — папка с результатами запросов к API:
  - Прямое геокодирование
  - Обратное геокодирование  
  [Перейти к файлам](https://github.com/myaccforedu-hash/geocoding-api-comparison-analysys/tree/main/result)

- `results-analysys.md` — итоговый аналитический отчет с таблицами, сравнением тарифов и выводами.  
  [Открыть отчет](https://github.com/myaccforedu-hash/geocoding-api-comparison-analysys/blob/main/results-analysys.md)

---

## Как использовать

1. Скачать репозиторий.  
2. Ознакомиться с датасетом адресов в `dataset-geo.xlsx`.  
3. Использовать файлы из папки `result/` для анализа точности и успешности геокодирования.  
4. Изучить подробный отчёт `results-analysys.md` для оценки возможностей сервисов и выбора подходящего API.  

---

## Вывод

Репозиторий демонстрирует, как различаются сервисы геокодирования по успешности прямого и обратного геокодирования, а также позволяет оценить тарифы и
возможности каждого сервиса. Дополнительное тестирование на реальных данных может помочь точнее выбрать сервис под конкретные задачи.
